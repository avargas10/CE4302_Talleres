--: Extensión SIMD: SSEx
Comandos para compilación

Dentro de la carpeta Taller2 ejecutar en consola:

-Para ejecutar hello_simd.c 

gcc hello_simd.c -msse4 -o hello_simd
./hello_simd

-Para ejecutar biggest.c el cual corresponde al ejercicio prático 2 

gcc biggest_simd.c -msse2 -o  biggest
./biggest

-Para ejecutar multiply.c el cual corresponde al ejercicio prático 3 

 gcc multiply.c -msse4 -o  multiply
 ./multiply
